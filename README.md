# POO Java

Repositório da disciplina de **Programação Orientada a Objetos com Java**, contendo exercícios, exemplos e projetos desenvolvidos durante as aulas.

---

## 📚 Conteúdo

Neste repositório você encontrará:

- Conceitos de POO
- Classes e Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Abstração
- Interfaces
- Tratamento de exceções
- Collections
- Exercícios práticos
- Projetos da disciplina

- Git e GitHub
